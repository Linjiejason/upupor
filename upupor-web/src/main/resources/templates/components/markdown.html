Markdown额外技巧
· 图片对齐
标准语法: ![描述](图片链接地址)
可设定图片高度宽度和对齐: ![描述#宽度#高度#对齐方式](图片链接地址)
设置图片大小(相对大小&绝对大小): ![描述#10%#50px](图片链接地址)
其中，`宽度`、`高度`支持：绝对像素值（比如200px）、相对外层容器百分比（比如50%）， `对齐方式`候选值有：左对齐（缺省默认）、右对齐（right）、居中（center）、悬浮左、右对齐（float-left/right）
· 插入目录
使用[[[[toc]]]],即可插入目录
